# Tài liệu tiếng Việt

Thư mục `docs/vi` đã được sắp xếp lại theo nhóm chủ đề để dễ tra cứu và mở rộng.

- `overview/`: tài liệu tổng quan sản phẩm, phạm vi và phân tích dự án
- `database-schema/`: tài liệu source of truth, phạm vi schema và quan hệ giữa các bảng
- `core-features/`: phân tích chi tiết các core flow
- `compilation/`: tài liệu tổng hợp, đồng bộ hóa và điểm vào để đọc nhanh
- `project-management/`: tài liệu workflow Scrum, Trello board và cách vận hành nhóm

Gợi ý thứ tự đọc:

1. `overview/PROJECT_BRIEF.md`
2. `overview/PROJECT_ANALYSIS.md`
3. `database-schema/DATABASE_SCHEMA_OVERVIEW_OF_MAIN_TABLES.md`
4. `database-schema/DIRECTION_AND_SCOPE_OF_THE_DATABASE_SCHEMA.md`
5. `core-features/DETAILED_ANALYSIS_OF_CORE_FEATURE_ORDER_VIA_QR_CODE.md`
6. `compilation/DOCUMENT-COMPILATION.md`

Tài liệu quản lý quy trình nhóm nằm riêng trong `project-management/` để không trộn với tài liệu nghiệp vụ và kỹ thuật.
