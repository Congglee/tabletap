# TableTap Dependency Version Guidance

When working on any task that involves both the front-end and back-end of the TableTap project in `tabletap-web` and `tabletap-server`, always check and pay close attention to the dependencies and their versions according to semantic versioning in `tabletap-server/package.json` and `tabletap-web/package.json`.

This helps ensure that you always stay aware of the libraries and versions currently used in the project, avoiding unnecessary issues caused by using incompatible or incorrect versions.

When searching for information, documentation, or solutions on the internet, prioritize sources that match the library versions currently used by the project to ensure compatibility and accuracy.
